INFRASTRUCTURE INTELLIGENCE PLATFORM

NHIP: National Highways Intelligence Platform

Helping NHAI identify overloaded vehicles and recover lost toll revenue.

A national-scale intelligence layer for India's toll network. The system fuses Weigh-in-Motion sensors, ANPR cameras, and FASTag transactions into a single vehicle record. It then applies automated overload detection designed to scale across India's highway network.

GitHub ↗

Role

Data Science Intern. Pipeline design, sensor fusion logic, and detection modelling.

Challenge

Three asynchronous sensor streams describing the same vehicle, none of them clean, none of them on time.

Solution

Streaming entity resolution joins WIM, ANPR and FASTag events per vehicle, feeding an overload classifier at toll speed.

Outcome

Overload flags raised while the truck is still at the plaza. An architecture built to extend nationwide.

PythonPySparkStructured Streaming Delta LakeMachine LearningWIM · ANPR · FASTag

SOFTWARE ARCHITECTURE VISUALIZATION

CodeMap

X-ray vision for codebases.

A GitHub repository analysis platform that parses a codebase and renders its software architecture as interactive dependency graphs. A new engineer can see how a project fits together before reading a single file.

GitHub ↗ Live Demo ↗

Role

Role: Full Stack Developer. From AST parsing in Python to the D3 rendering layer.

Challenge

Real dependency graphs are hairballs; hundreds of modules render as noise, not insight.

Solution

Hierarchical clustering by package plus force-directed layout to collapse what doesn't matter and surface what does.

Outcome

An architecture view that makes day-one onboarding to an unfamiliar repository measurably faster.
